# Лаб.3 - Frequency-Based Decryption

В основі шифру Цезаря лежить операція зсуву. Тому принципіальної різниці між шифруванням та дешифруванням нема: змінються лише знак числа на яке відбувається зсув, тобто використовується одна і та сама функція. Для знаходження початкового повідомлення без знання зміщенні необхідно перебрати всього 26 варіантів. Для того, щоб алгоритмічно вирахувати правільний варіант використовується таблиця частот зустрічання літер алфвавіту. На основі цього обрховується "рахунок" кожного варіанту. Найбільший "рахунок" відповідає правильному початковому повідомленню.

```
# python3 lab03/lab03.py

Оригінальний текст:        Lorem ipsum dolor sit amet, consectetur adipiscing elit
Зашифрований текст:        Ehkxf bilnf whehk lbm tfxm, vhglxvmxmnk twbiblvbgz xebm
Розшифрований текст:       Lorem ipsum dolor sit amet, consectetur adipiscing elit
Розшифрований текст (хак): Lorem ipsum dolor sit amet, consectetur adipiscing elit
```
