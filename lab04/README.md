# Лаб.4 - Hacking at RobberCity

Ідея реалізації дуже проста: окільки операція XOR є комутативною (`A ^ B = B ^ A`) та асоціативною (`(A ^ B) ^ C = A ^ (B ^ C)`), то ми можемо відновити початкове повідомлення, маючи лише три перехоплені шифротексти зробвиши XOR між ними.

```
# python3 lab04/lab04.py

Оригінальний текст:        Lorem ipsum dolor sit amet
Крок 1 (Аліса -> Боб):     2d031b06087f1a15100708543b040916134c1a0a117f12080606
Крок 2 (Боб -> Аліса):     4f6c79597b1a79677573573f5e7d6b797113696f720d777c596d
Крок 3 (Аліса -> Боб):     2e00103a1e450a021601324b01160e00107f000c175204193a1f
Крок 4 (Боб читає):        Lorem ipsum dolor sit amet
Розшифрований текст (хак): Lorem ipsum dolor sit amet
```
